# 🧠 DealMind AI  
### Memory-Powered Sales Intelligence Agent

## 🚀 Project Overview

DealMind AI is an intelligent sales assistant that helps sales teams close deals faster by combining real-world deal data, persistent memory (Hindsight), and high-performance LLMs (Groq 32B & 120B).

---

## 🎯 Problem Statement

Sales teams face challenges such as forgetting past interactions, re-reading CRM notes, inconsistent objection handling, and losing deals due to poor follow-up.

Goal: Build an AI agent that remembers interactions, understands deal context, and suggests winning strategies.

## 🔁 CascadeFlow

User Input → Objection Detection → Memory Recall → Dataset Context → Strategy → Next Action → LLM → Response → Memory Save

---

## 🔄 Workflow

1. User message
2. Detect objection
3. Load deal data
4. Recall memory
5. Generate strategy
6. Generate response
7. Save memory
